# Nuestro Primer Año 💖

Libro mágico interactivo: un año de recuerdos en fotos y una carta.

## Cómo ver el proyecto

- **Desarrollo:** `npm install` y luego `npm run dev`
- **Build:** `npm run build`

## Publicar en GitHub Pages

1. En el repo: **Settings → Pages**
2. En "Build and deployment", **Source:** elige **GitHub Actions**
3. Tras cada push a `main`, el workflow construye y publica la web

La página quedará en: **https://jackss0.github.io/personal-web/**

## Música

Opcional: coloca un archivo `song.mp3` en `public/music/` para la música de fondo en la galería.
